<div>

La etiqueta <div> define una división o una sección en un documento HTML. Se utiliza para agrupar elementos en bloque y darles formato con CSS.

Esto es un div

<span>

La etiqueta <span> se utiliza para agrupar elementos en línea en un documento.

Esto es un span

<strong>

La etiqueta <strong> se utiliza para definir texto con gran importancia. El contenido dentro de esta etiqueta se muestra típicamente en negrita.

Este texto es un strong

<i>

La etiqueta <i> se utiliza para definir una parte del texto en una voz o estado de ánimo diferente. El contenido dentro de esta etiqueta se muestra típicamente en cursiva.

Este texto está en cursiva

<u>

La etiqueta <u> se utiliza para subrayar texto.

Este texto está subrayado

<b>

La etiqueta <b> se utiliza para llamar la atención sobre el texto sin implicar una importancia añadida o énfasis.

Este texto está en negrita

<strike>

La etiqueta <strike> se utiliza para mostrar el texto con una línea que lo atraviesa.

Este texto está tachado

<small>

La etiqueta <small> se utiliza para definir texto más pequeño.

Este texto es small

<em>

La etiqueta <em> se utiliza para enfatizar texto. El contenido dentro de esta etiqueta se muestra típicamente en cursiva.

Este texto está enfatizado

<header>

La etiqueta <header> representa un contenedor para contenido introductorio o un conjunto de enlaces de navegación.

Esto es un header

<section>

La etiqueta <section> define una sección en un documento.

Esto es una section

<article>

La etiqueta <article> especifica contenido independiente y auto-contenido.

Esto es un article

<aside>

La etiqueta <aside> define contenido aparte del contenido principal (como una barra lateral).

Esto es un aside

<footer>

La etiqueta <footer> define un pie de página para un documento o sección.

Esto es un pie de página

<nav>

La etiqueta <nav> define un conjunto de enlaces de navegación.

Esto es una sección de navegación

<img>

La etiqueta <img> se utiliza para incrustar una imagen en una página HTML.

<br>

La etiqueta <br> inserta un salto de línea único.

Línea uno
Línea dos

<hr>

La etiqueta <hr> define un cambio temático en una página HTML (por ejemplo, un cambio de tema).

<form>

La etiqueta <form> se utiliza para crear un formulario HTML para la entrada del usuario.
